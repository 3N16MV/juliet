This is the distribution package for Juliet Initial Release (April, 2024).

This package is Exercise 4 of Chapter 8 from the Book: Python for 
Everybody: Exploring Data in Python 3 by Charles Severance. 
ISBN: 9781530051120

This exercise was conducted as an assignment for Cyebrsecurity Class
CYB333-25957 at National University. The purpose of this assignment
is to find all unique words in a file and order them alphabetically.
You can find a copy of the text file at www.py4e.com/code3/romeo.txt.
